Part 4

Users can enter links in the text editor by using tags. For example, “<a>This is a link </a>”. When the user clicks “Done”, the editor should detect if the user has entered a link tag in the editor. If so, add that text as an actual link (like this: This is a link) to the bottom of the page (outside of the text editor). The links at the bottom of the page should be colored red and blue alternatively, and they do not need to actually link to another site.
